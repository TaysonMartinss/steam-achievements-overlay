Para começar, você precisa configurar sua chave da API Steam. É rápido e gratuito!

Como obter sua API Key:

  1. Acesse: https://steamcommunity.com/dev/apikey
  2. Faça login na sua conta Steam
  3. No campo "Domain Name", digite: localhost
  4. Aceite os termos e clique em "Register"
  5. Copie a chave gerada (sequência de 32 caracteres)

Sua privacidade está protegida: A API Key é armazenada apenas localmente no seu computador e é utilizada exclusivamente para buscar suas informações públicas da Steam.
